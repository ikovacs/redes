Para poder utilizar:
	el traceroute, se deberá escribir el siguiente comando: 
		sudo python trace.py (URL a analizar)
	
	la simulación de la conexión:
		sudo python ping.py (URL a analizar)

En el caso de querer modificarse la cantidad de paquetes que se envían para cada escenario, se deberá modificar directamente en el código.
